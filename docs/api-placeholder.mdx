The API reference is generated from the ZeroQuarry OpenAPI specification.

Run:

```bash
npm run gen:api
```

The generated pages are written to `docs/api` and are intentionally ignored by
Git. The source specification is copied from
`../security-bug-finder/static/ZeroQuarry.spec.yml` during `npm run start` and
`npm run build`.
